学法语的学霸看过来--一步步教你用python编自己的法语词典

发表于 2016-07-29   |   分类于 python ， 编程   |     |   阅读次数   |  

记得在当时学法语时，非常蛋疼，尤其是没有一个趁手的法语词典记录工具，虽说法语助手查词的功能很强大，但是单词本的功能有点鸡肋，没法本地导出，也没法本地记录查过的单词发音文件。所以一直想开发属于自己的法语词典。但当时，自己编程水平不足，而且也忙东忙西，一直没有付诸实施。现在工作了，法国公司工作非常轻松，有大把的时间去提升自己的编程能力，终于，借助python和爬虫的相关知识，建立了属于自己的法语词典。

这个词典相比于法语助手的特色是：

  1. 释义一目了然，没有其他广告
  2. 单词本功能，自动把查询过的单词保存到本地，并在本地保存对应的单词发音，适合查询后复习
  3. 可拓展性强，比如：可以将查询过得单词及其释义保存在两个文件里面，可以只看单词回想释义，或者看释义回想单词，方便单词的学习。再比如：可以编写基于单词本的单词背诵功能
  4. 终生免费开源

好了，下面一步步教大家如何建立自己的法语词典。

要想读懂本文，最好懂一点python，而且有geek精神，基本的编程思想还是要有的。
